# 天和文档

## 软件介绍

这个软件我们的界面思路仿照了Office，不过我们把字体文档等组件放在了最上面，把特殊组件放在了左边（可以自行调整），如果你懒得移动鼠标，还可以按下 / 按键，使用方向键来选择  

### 目前支持格式

#### 读取

.docx 
.doc 
.pdf 
.png 
.md 
.txt 

#### 写入

.docx 
.doc 
.md 
.txt 
[hello](#整体介绍)

## 整体介绍

这是一个文档类软件，虽然不如市场上其他软件，但我们保证它永远开源（目前免费）

这款软件我们对macOS和Windows版本使用了不同的开发框架，Windows使用QT编写，而macOS使用swiftUI编写
